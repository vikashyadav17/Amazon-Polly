What is Amazon Polly?

Amazon Polly is a cloud service that converts text into lifelike speech, enabling developers to create applications that talk. Use the tools below to interact with Amazon Polly's features.

Features

  • Text-to-Speech Conversion

    Quickly convert written content into lifelike speech.

  • Multiple Languages

    Supports a variety of languages and accents.

  • Custom Lexicons

    Define how specific words are pronounced.
